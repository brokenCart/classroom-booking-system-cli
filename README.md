# 🏫 Classroom Booking System
A Python command-line application for managing classroom scheduling with room creation, availability filtering, booking management, and CSV-based data persistence.

## Features
- Create new rooms
- Search available rooms by building, capacity, and hour
- Book rooms for specific timeslots
- View room schedules
- Persistent storage using CSV
- Custom error handling

## Installation
```bash
git clone https://github.com/brokenCart/classroom_booking_system_cli.git
cd classroom_booking_system_cli
```

## Run
```bash
python main.py
```
